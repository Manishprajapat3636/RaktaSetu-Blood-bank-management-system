# RaktaSetu-Blood-bank-management-system
RaktaSetu is a web-based Blood Bank Management System designed to bridge the gap between blood donors, blood banks, and patients in need. The platform simplifies blood availability tracking, donor management, and emergency blood requests through a centralized and user-friendly system.

The application enables users to search for blood groups, locate nearby blood banks, and connect with eligible donors in real time. Blood banks can manage donor records, blood stock levels, and request histories efficiently, ensuring faster response during emergencies.

# Key Features
User registration and authentication (Donor / Blood Bank)

1.Real-time blood availability tracking
2.Search blood by group and location
3.Donor and blood bank profile management
4.Request and notification system for urgent blood needs
5.Secure and responsive web interface

# Tech Stack

Frontend: HTML, CSS, JavaScript / React

Backend: Node.js, Express.js

Database: MongoDB

Authentication: JWT / Session-based Auth

# Purpose

RaktaSetu aims to create a reliable digital platform that improves blood accessibility, reduces response time in critical situations, and supports healthcare services by connecting donors with those in need efficiently.
